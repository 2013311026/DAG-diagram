前端实现有向无环图
===========
`基于SVG的DAG(单向无环图)前端实现.适用于机器学习平台或流程图.`

本项目为公司内部PAI需求,
参考了国内外其他公司的DAG实现

* 1. 图. 节点. 连线 的展示
* 2. 节点的增删, 位置变动 ( 开放右键功能
* 3. 连线的增删 ( 开放右键功能
* 4. 整图放缩 拖动 框选节点
